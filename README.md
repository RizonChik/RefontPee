# RefontPee

Небольшой забавный плагин для Spigot: команда `/pee` включает эффект "писания" с желтым градиентом частиц и звуками воды.

## Возможности
- `/pee` включает режим на несколько секунд и сам выключается
- частицы направлены по взгляду игрока
- градиентные сообщения на русском
- реалистичные водные звуки

## Требования
- Minecraft/Spigot 1.16.5
- Java 8+
- Maven

## Установка
1. Соберите плагин:
   ```bash
   mvn package
   ```
2. Файл `target/RefontPee.jar` положите в папку `plugins` вашего сервера.
3. Перезапустите сервер или выполните `/reload`.

## Использование
- В игре напишите `/pee` — эффект включится на короткое время.

## Настройка
Параметры находятся в коде:
- длительность эффекта
- частота частиц
- сила звуков
